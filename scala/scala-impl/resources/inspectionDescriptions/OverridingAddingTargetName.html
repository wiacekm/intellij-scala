Reports the definitions with a @targetName annotation that override definitions without @targetName annotation.

See The @targetName annotation.

The quick-fix removes @targetName annotation.

Example:


  class A:
    def foo: Int = 1

  class B extends A:
    @targetName("overriddenFoo")
    override def foo: Int = 2

After the quick-fix is applied:


  class A:
    def foo: Int = 1

  class B extends A:
    override def foo: Int = 2

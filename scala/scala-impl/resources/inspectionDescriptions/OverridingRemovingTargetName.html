Reports the definitions without a @targetName annotation that override definitions with @targetName annotation.

See The @targetName annotation.

The quick-fix adds @targetName annotation with external name of the overridden definition.

Example:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    override def *(i: Int): A = ???

After the quick-fix is applied:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???

Reports the definitions with a @targetName annotation that override definitions with @targetName annotation but have different external names.

See The @targetName annotation.

The quick-fix replaces the @targetName annotation's external name with external name of the overridden definition.

Example:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("times")
    override def *(i: Int): A = ???

After the quick-fix is applied:


  trait A:
    @targetName("multiply")
    def *(i: Int): A

  class B extends A:
    @targetName("multiply")
    override def *(i: Int): A = ???

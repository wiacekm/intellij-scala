Reports the usage of @targetName annotation on multiple targets.

The quick-fix removes @targetName annotation.

Example:


  // 1
  val list: List[Int] = (1 to 5).toList
  @targetName("head_and_tail")
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    @targetName("b_and_c")
    case B, C
    case D

After the quick-fix is applied:


  // 1
  val list: List[Int] = (1 to 5).toList
  val (head, tail) = (list.head, list.tail)

  // 2
  enum E:
    case A
    case B, C
    case D
